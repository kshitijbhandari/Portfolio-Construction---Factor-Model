# 🎯 NEXT STEPS - Get Your Streamlit App Running

## The Fastest Way (Windows)

### Step 1: Run the Quick Start Script
```bash
cd c:\Users\kshit\Personal_Factor_model
quickstart.bat
```

This will:
1. ✅ Check Python installation
2. ✅ Create virtual environment
3. ✅ Install all dependencies
4. ✅ Extract functions from notebook
5. ✅ Start the Streamlit server

**That's it!** Your app will open at http://localhost:8501

---

## If Quick Start Doesn't Work

### Option A: Manual Setup (5 minutes)

```bash
# 1. Go to project directory
cd c:\Users\kshit\Personal_Factor_model

# 2. Create virtual environment
python -m venv venv
venv\Scripts\activate.bat

# 3. Install dependencies
pip install -r requirements.txt

# 4. Extract functions from notebook
python setup_streamlit.py

# 5. Run app
streamlit run app.py
```

### Option B: Manual Function Extraction (10 minutes)

If `setup_streamlit.py` doesn't create `utils.py`:

```bash
# 1. Install nbconvert
pip install nbconvert

# 2. Convert notebook to Python
jupyter nbconvert --to script model.ipynb --output utils.py

# 3. Run app
streamlit run app.py
```

## 🔧 Troubleshooting

### Issue: "Python not found"
- Install from: https://www.python.org/downloads/
- **Important**: Check "Add Python to PATH" during installation

### Issue: "ModuleNotFoundError: No module named 'utils'"
- Run: `python setup_streamlit.py`
- Or see EXTRACT_FUNCTIONS.md for manual extraction

### Issue: "Port 8501 is already in use"
- Kill the process: `netstat -ano | findstr :8501`
- Or start on different port: `streamlit run app.py --server.port 8502`

### Issue: "Data files not found"
- Ensure these CSV files are in the same directory:
  - ✅ nifty_stocks_data (1).csv
  - ✅ nifty50_index_data.csv
  - ✅ FF_Nifty50.csv
  - ✅ Nifty_50.csv
